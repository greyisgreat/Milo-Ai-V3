MILO - Machine Intelligence Liaison Operator
MILO is a JARVIS-inspired AI desktop assistant designed for task automation, self-healing code generation, and interactive visualization. It utilizes the Google Gemini API for its intelligence and runs local processes for speech and system operations.

launch milo by opening codespaces and typing 2 commands 
1. Install your tools
Copy and paste this command into your terminal and press Enter to make sure everything needed to run the app is installed:

Bash
pip install PySide6 psutil google-generativeai SpeechRecognition pyttsx3 python-dotenv
2. Launch M.I.L.O
Once that finishes installing, type this command and press Enter:

Bash
python main.py
What will happen next:

The terminal will immediately show: Please paste your Gemini API Key here and press Enter:.

You paste your key there, press Enter, and the application will initialize.
